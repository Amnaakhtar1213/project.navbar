## NAVBAR

This project is about navbar. I build it with **HTML**, **CSS** and **JavaScript**.It has a transparent blur navbar which has a links .When we hover on links it has a good transition and border appearance .It has a responsiveness that appears good on different devices.Its Feature includes a clean dropdown and toggle button for smaller screens.

--- 

## fEATURES 

1. Responsive design for all screen sizes.
2. Beautifull appearance when we hover on links.
3. Mobile friendly toggle button menu.
4. Smooth dropdown animation for links.
